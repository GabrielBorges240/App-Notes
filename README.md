# Notes App

Aplicativo simples para criação e organização de anotações.

Demo:  https://gabrielborges240.github.io/App-Notes/

## Funcionalidades
- Criar, editar e excluir notas
- Armazenamento local
- Interface limpa e organizada

## Tecnologias
- HTML5
- CSS3
- JavaScript
- LocalStorage

## Como executar
Abra o arquivo `index.html` no navegador.
